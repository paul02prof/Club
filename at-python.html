Qu'est-ce que Python ?

Python est un langage de programmation interprété, haut niveau et généraliste. Créé par Guido van Rossum et sorti en 1991, il est devenu l'un des langages les plus populaires au monde.

Sa philosophie met l'accent sur la lisibilité du code avec sa syntaxe claire et sa gestion automatique de la mémoire. Python est utilisé dans de nombreux domaines : développement web, science des données, intelligence artificielle, automatisation, et bien plus.

Pourquoi choisir Python ?

Facile à apprendre

La syntaxe claire et intuitive de Python le rend idéal pour les débutants en programmation.

Écosystème riche

Des milliers de bibliothèques et frameworks pour tous les besoins : Django, Flask, NumPy, Pandas, TensorFlow...

Polyvalent

Développement web, analyse de données, IA, scripts d'automatisation, jeux vidéo... Python fait tout !

Fonction simple

def calculer_carre(x):
    return x * x

# Appel de la fonction
resultat = calculer_carre(4)
print(resultat)  # Affiche 16

Liste en compréhension

# Créer une liste des carrés des nombres pairs de 0 à 10
carres_pairs = [x*x for x in range(11) if x % 2 == 0]
print(carres_pairs)  # [0, 4, 16, 36, 64, 100]
